PintoStack is a new approach to running and managing distributed systems. PintoStack gives you immutable container infrastructure, with service discovery and continuous logging.

PintoStack is as easy as 1,2,3:

1

Select your destination: public or private cloud or local machine

2

Build your app or service container

3

Run it at scale using resources available

Congratulations, you have set-up your system for enterprise computing and/or data processing!

PintoStack components:

Docker Containers
Immutable infrastructure
with Ansible and Vagrant
Fault tolerant and fully elastic
Marathon framework
Dynamic service discovery
via Consul
Centralized logging with
ELK stack
iPython Notebook with
Apache Spark for big data jobs
